# HMS Unity IAP
HMS Unity(2019) Inn-App Purchasing 

# Preconditions
1. Huawei Account activated
2. Merchant Account activated
3. In App purchasing API enabled
4. Add Products to AppGallery Connect

Drag and drop this source to your asset folder.

# Required Configurations
1. Add keystore file
  `Build Settings -> Player Settings -> Publishing Settings -> Add keystore file`
  `Sign and get -> SHA256 keytool -list -v -keystore yourkeyfile.keystore`
2. Add Agconnect-service.json file
3. Configure AndroidManifest file
